#include "common_options.txt"
.TH GRDCLIP 1 MANDATE GMT_STRING "Generic Mapping Tools"
.SH NAME
grdclip \- Clip the range of grids
.SH SYNOPSIS
BD(grdclip) IT(ingrid) OPT(G)IT(outgrid) [ GMT_Rgeo_OPT ] 
[ OPT(Sa)IT(high/above) ] [ OPT(Sb)IT(low/below) ] [ OPT(Si)IT(low/high/between) ] [ OPT(Sr)IT(old/new) ] 
[ GMT_V_OPT ] 
.SH DESCRIPTION
BD(grdclip) will set values < IT(low) to IT(below) and/or values > IT(high) to IT(above).  You
can also specify one or more intervals where all values should be reset to IT(between), or individual
values to replace. Such operations are useful
when you want all of a continent or an ocean to fall into one color or gray shade in image processing,
when clipping of the range of data values is required, or for reclassification of data values. 
The IT(above), IT(below), IT(between), IT(old) and IT(new) can be any number or even NaN (Not a Number).
You must choose at least one of the OPT(S) options.  Use OPT(R) to only extract a subset of the IT(ingrid) file.
#include "explain_commonitems.txt"
.SH REQUIRED ARGUMENTS
.TP
IT(ingrid)
The input 2-D binary grid file.
.TP
OPT(G)IT(outgrid)
IT(outgrid) is the modified output grid file.
.SH OPTIONAL ARGUMENTS
#include "explain_-R.txt" 
Using the OPT(R) option will select a subsection of IT(ingrid) grid. If this subsection exceeds the
boundaries of the grid, only the common region will be extracted.
.TP
OPT(Sa)IT(high/above)
Set all data[i] > IT(high) to IT(above).
.TP
OPT(Sb)IT(low/below)
Set all data[i] < IT(low) to IT(below).
.TP
OPT(Si)IT(low/high/between)
Set all data[i] >= IT(low) and data[i] <= high to IT(between).  Repeat the option
for as many intervals as are needed.
.TP
OPT(Sr)IT(old/new)
Set all data[i] == IT(low) to IT(new).  This is mostly useful when
  your data are known to be integer values.  Repeat the option
for as many replacements as are needed.

.SH EXAMPLES
To set all values > 70 to NaN and all values < 0 to 0 in file data.nc:
.sp
grdclip data.nc -Gnew_data.nc -Sa70/NaN -Sb0/0 -V
.br
.sp
To reclassify all values in the 25-30 range to 99, those in 35-39 to 55, exchange 17 for 11 and all values < 10 to 0 in file classes.nc, try
.sp
grdclip classes.nc -Gnew_classes.nc -Si25/30/99 -Si35/39/55 -Sr17/11 -Sb10/0 -V
