#include "common_options.txt"
.TH GRDCUT 1 MANDATE GMT_STRING "Generic Mapping Tools"
.SH NAME
grdcut \- Extract subregion from a grid
.SH SYNOPSIS
BD(grdcut) IT(ingrid) OPT(G)IT(outgrid) GMT_Rgeo_OPT [ OPT(S)[BD(n)]IT(lon/lat/radius)[IT(unit)] ] 
[ GMT_V_OPT ] [ OPT(Z)[BD(n)]IT(min/max)] ] [ GMT_f_OPT ]
.SH DESCRIPTION
BD(grdcut) will produce a new IT(outgrid) file which is a subregion of IT(ingrid).
The subregion is specified with OPT(R) as in other programs; the
specified range must not exceed the range of IT(ingrid).  If in doubt, run BD(grdinfo)
to check range.  Alternatively, define the subregion indirectly via a range check on the node values
or via distances from a given point.
Complementary to BD(grdcut) there is BD(grdpaste), which will join together two
grid files along a common edge.
#include "explain_commonitems.txt"
.SH REQUIRED ARGUMENTS
.TP
IT(ingrid)
This is the input grid file.
.TP
OPT(G)IT(outgrid)
This is the output grid file.
.SH OPTIONAL ARGUMENTS
#include "explain_-R.txt"
This defines the subregion to be cut out.
.TP
OPT(S)[BD(n)]IT(lon/lat/radius)[IT(unit)]
Specify an origin and radius; append a distance unit (see UNITS) and we determine the corresponding rectangular region so that
all grid nodes on or inside the circle are contained in the subset.  If OPT(Sn) is used we set all nodes outside the circle to NaN.
#include "explain_-V.txt"
.TP
OPT(Z)[BD(n)]IT(min/max)]
Determine the new rectangular region so that all nodes outside this region are also outside the given IT(z)-range [-inf/+inf].
To indicate no limit on min or max, specify a hyphen (-).  Normally, any NaNs encountered are simply skipped.  Use
OPT(Zn) to consider a NaN to be outside the IT(z)-range.

.SH EXAMPLES
Suppose you have used BD(surface) to grid ship gravity in the region between 148E - 162E and 8N - 32N, and you
do not trust the gridding near the edges, so you want to keep only the area between 150E - 160E and 10N - 30N, then:
.sp
grdcut grav_148_162_8_32.nc -Ggrav_150_160_10_30.nc -R150/160/10/30 -V
.sp
To return the subregion of a grid such that any boundary strips where all values are entirely
above 0, try
.sp
grdcut bathy.nc -Gtrimmed_bathy.nc -Z-/0 -V
.sp
To return the subregion of a grid that contains all nodes within a distance of 500 km from the point 45,30 try
.sp
grdcut bathy.nc -Gsubset_bathy.nc -S45/30/500k -V
