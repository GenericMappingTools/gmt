GMT for Windows


We offer four Windows Installers for those who do not wish to compile from source:

Required: Either the 32-bit or 64-bit GMT installer. Will install all executables (including the supplements), the netCDF and GDAL DLL libraries, gawk, the crude, low, and intermediate GSHHS v2.1.1 coastline data, all documentation in HTML format, and the example batch scripts and test data [17 Mb].

Optional 1: The GSHHS installer will install the optional high and full resolution GSHHS v2.1.1 coastlines [37 Mb].
Optional 2: The PDF installer will install the documentation in PDF format [15 Mb].

Should you have additional needs for source code or tutorial data sets, etc., you may download the complete GMT distribution as bzipped tarballs from any of our GMT ftp sites and extract the files with a suitable extractor (such as 7-zip).
