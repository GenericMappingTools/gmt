TTT4 Ideas to pursue: OpenMP

Instead of switch (n_nodes):
	if (ok_to_use[64]) ss = SS_064, tt_inc[64] = GET_TT_INC (d34y[j0]);

Need to make this into a loop than can be OpenMP'd.  So

build inline functions func_SS_xxx, e.g.

static inline double func_SS_064 (float *s, double *d, TTT_LONG ij, TTT_LONG *p, TTT_LONG j0) {
	double ss = SS_064;	/* Uses macro to get normalized slowness, needs s[], ij, and p[] */
	return GET_TT_INC (d[j0]);	/* Uses ss and this distance */
}
and create array of function pointers to these, i.e. (build these via awk as include files)

ss_funct[64] = func_SS_064;
...

will need corresponding array of pointers *d so that
d[64] = d[65] = d[66] = d[67] = d34y;	/* Etc for all the others */

Then replace loop over nodes with two loops, one using OpenMP

#ifdef _OPENMP
#pragma omp parallel for private(k) shared(n_nodes,ok_to_use,ss_func,s,d,ij,p,j0,tt_inc)
#endif 
for (k = 0; k < n_nodes; k++) {	/* Compute increment in travel times to all surrounding nodes */
	if (ok_to_use[k]) tt_inc[k] = ss_func[k] (s, d[k], ij, p, j0);
}
